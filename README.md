# 水電Gas記帳

Python/Tkinter 桌面記帳工具，用來記錄自來水、電力、瓦斯費用與度數，並整合台灣統一發票自動對獎。

## 目前版本

`V0.1.7`

主程式：

```bash
python3 WPG_V0.1.7.py
```

## 功能

- 自來水、電力、瓦斯三個記帳分頁
- 年份使用中華民國紀年
- 月份、費用、計價度數、去年同期度數、平均度數、發票號碼記錄
- 輸入後自動儲存
- 依年份與月份重新排序
- 回復上一步
- 費用與平均度數折線圖
- 計價度數大於去年同期度數時以紅色標示
- 統一發票對獎分頁
- 開啟程式時會自動重試空白或「未對獎」記錄
